**Android:** Termuxu açın və bu kodu yapışdırın: `bash <(curl -L https://git.io/Jklqq))`

**iOS:** iSH açın ve bu kodu yapışdırın: `apk update && apk add bash && apk add curl && curl -L -o dto-ios.sh https://git.io/JklX4 && chmod +x dto-ios.sh && bash dto-ios.sh`
